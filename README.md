# React Design Patterns 

We can identify different React Design Patterns.

They are,
- HOC / Higher Order Components Pattern
- Factory Pattern
- Provider Pattern
- Custom Hook Pattern
- Render Props Pattern
- Compound Pattern
- Control and Uncontroll Components Pattern
- Container Presenter Pattern
